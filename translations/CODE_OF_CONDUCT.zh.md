# 行为准则

## 我们的承诺

作为成员、贡献者和维护者,我们承诺让每个人参与我们的社区都成为不受骚扰的体验,无论其年龄、体型、可见或不可见的残障、族裔、性别特征、性别认同与表达、经验水平、教育、社会经济地位、国籍、外貌、种族、宗教,或性取向与性别认同。

## 我们的标准

有助于营造积极环境的行为示例:

- 对新人和首次贡献者友善、欢迎
- 尊重不同的意见、观点和经验
- 给予并优雅地接受建设性反馈
- 专注于对社区最有利的事

不可接受的行为示例:

- 公开或私下的骚扰、侮辱或贬损性言论
- 未经明确许可公开他人的私人信息
- 挑衅,或持续扰乱讨论
- 任何在职业场合中被合理认为不当的行为

## 执行

辱骂、骚扰或其他不可接受的行为,可通过 [Discord](https://discord.gg/gqhDWfZbgU) 或发送至 `conduct@zetic.ai` 向维护者举报。所有投诉都会得到及时、公正的审查与调查。维护者有义务尊重任何事件举报者的隐私与安全。

## 出处

本行为准则改编自 [Contributor Covenant](https://www.contributor-covenant.org) 2.1 版。
